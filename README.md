# Movie Library

**Автор:** Калмыкова Анна Ивановна

## Описание
Movie Library — это GUI-приложение на Python (Tkinter) для хранения информации о фильмах. Программа позволяет добавлять фильмы, просматривать их в таблице, фильтровать по жанру и году выпуска, а также сохранять и загружать данные через JSON.

## Возможности
- Поля: название, жанр, год выпуска, рейтинг.
- Добавление фильма в таблицу.
- Фильтрация по жанру и году выпуска.
- Проверка ввода:
  - год должен быть числом;
  - рейтинг должен быть от 0 до 10.
- Сохранение и загрузка JSON.
- Удаление выбранного фильма.

## Как использовать
1. Установите Python 3.
2. Запустите файл `movie_library.py`.
3. Введите название, выберите жанр, укажите год и рейтинг.
4. Нажмите кнопку **Добавить фильм**.
5. Используйте фильтр по жанру или году.
6. Сохраняйте и загружайте коллекцию через JSON-кнопки.

## Примеры тестов
- Добавить фильм с годом `2024` и рейтингом `8.5`.
- Ввести год `abcd` — программа должна показать ошибку.
- Ввести рейтинг `12` — программа должна показать ошибку.
- Отфильтровать фильмы по жанру `Драма`.
- Сохранить коллекцию в JSON и загрузить её обратно.
